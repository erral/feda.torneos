feda.torneos Installation
-------------------------

To install feda.torneos using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``feda.torneos`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        feda.torneos
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
